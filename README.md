## 📂 Repository Structure

Yeh repo step-by-step Pandas concepts cover karta hai:

* `01_importation.ipynb` → Pandas import karna
* `02_read_data.ipynb` → CSV/Data file read karna
* `03_head_&_tail.ipynb` → Top aur bottom records dekhna
* `04_describe_datafile.ipynb` → Statistical summary generate karna
* `05_info_datafile.ipynb` → Dataset ki information check karna
* `06_dataselection.ipynb` → Specific columns select karna
* `07_dataselect_by_row.ipynb` → Rows select karna
* `08_fill_null_cells.ipynb` → Missing values handle karna
* `09_rename_column.ipynb` → Columns rename karna
* `10_change_datatype.ipynb` → Data types change karna
* `11_NewCol_calu_export.ipynb` → New column create karna, calculations aur export
* `12_merge_2_df.ipynb` → Multiple DataFrames merge aur concat karna
